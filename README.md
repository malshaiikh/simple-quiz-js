# Simple Quiz JS

A small interactive quiz game built with HTML, CSS, and JavaScript.  
Test your knowledge with fun multiple-choice questions, track your score, and try to beat your high score!

## Features

- Multiple-choice questions with instant feedback
- Highlights correct and incorrect answers
- Dynamic progress bar for visual progress tracking
- Randomized question order for a different experience each time
- Tracks high score using local storage=

## How to Play

1. Click **Start Quiz** to begin.
2. Select an answer for each question.
3. See your score and progress update in real-time.
4. View your final score and try to beat your high score!
5. Click **Restart Quiz** to play again.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
